# Web Interface Guidelines

Review UI code against the latest Web Interface Guidelines.

## Quick Workflow

1. Fetch the current guidelines.
2. Review the specified files or patterns.
3. Flag only issues that materially affect usability, accessibility, or clarity.
4. Output findings in the format required by the guidelines.

## Non-Negotiables

- Always fetch the latest guidelines before review.
- Follow the output format in the fetched rules.
- Ask for file targets if none are provided.
